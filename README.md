🧠 Mental Engine — Cognitive Load Simulation System

A modular and extensible Python engine that simulates human cognitive state evolution under workload and recovery cycles.

This project models how energy, stress, resilience, and clarity evolve over time based on activity intensity and recovery, with built-in burnout detection and prediction.

🚀 Features
🔁 Time-based cognitive state simulation
🧮 Non-linear fatigue modeling (golden ratio exponent)
🌙 Circadian rhythm integration
🔥 Burnout detection system
🔮 Burnout prediction (forward simulation)
🧩 Strategy pattern (plug your own simulation logic)
📊 CLI visualization
🧼 Clean architecture (DDD-inspired)
